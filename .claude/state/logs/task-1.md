### 발견한 문제점

1. **components/index.tsx**
   - 모든 UI 컴포넌트가 구분 없이 한 파일에 뭉쳐있음
   - Button, Card, Dialog, Select, Table, Input, Textarea 등이 하나의 파일에 존재

2. **PostsManagerPage.tsx**
   - 로컬 상태로 대부분 관리됨
   - 컴포넌트 분리가 안되어 있음
   - API 호출, 상태 관리, UI 렌더링 로직이 모두 뭉쳐있음
   - 매우 높은 결합도와 낮은 응집도

3. **재사용 모듈 부재**
   - type 정의가 분리되어 있지 않음
   - util 등 재사용 가능한 모듈이 보이지 않음
